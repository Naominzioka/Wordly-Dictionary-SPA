# Wordly Dictionary SPA

A simple single-page app to search for word definitions, examples, and synonyms.

## What It Does

- Search for any word
- Get the definition
- See example usage
- Find synonyms

## How to Use

1. Open `single-page-app.html` in your browser
2. Type a word in the search box
3. Click Submit
4. View the word details below

## Features

- Fetches data from [Dictionary API](https://dictionaryapi.dev/)
- Clean, simple design
- Shows part of speech, definitions, examples, and synonyms

## Files

- `single-page-app.html` - The page structure
- `single-page-app.js` - The app logic (fetches and displays data)
- `single-page-app.css` - The styling
- `render-data.js` - Extra functions for rendering (optional)

## Requirements

Just a web browser! No installation needed.
